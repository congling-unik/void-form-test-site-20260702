RETURNS

Returns

An order request is not a paid or confirmed order, so it does not create a return or refund transaction.

Before placing an order

Return eligibility, timing, condition requirements, costs, exclusions, and the refund route will be shown before paid orders become available.

Order issues

Customer care will assess concerns such as transit damage, an incorrect item, or a missing item against the terms shown with the order.

Refund route

Any future eligible refund will follow the payment route and terms presented with a paid order. No payment is currently collected on this site.

Customer care

For product and care questions, contact us through Instagram. For future order support, include the order number.
